
The cluster auditor (like many other BigBang components) allows you to specify additional network controls through values provided to the chart. The folders for networkPolicies for this repo can be found [here](https://repo1.dso.mil/big-bang/product/maintained/cluster-auditor/-/tree/main/chart/templates/bigbang?ref_type=heads). Please consult [the Big Bang User Guide](https://docs-bigbang.dso.mil/latest/docs/guides/using-bigbang/network-policies/) for information and instruction in this regard.
